Esto es una aplicacion de chat que fue diseñada unicamente para implementar y ejemplificar el uso de los websockets, la aplicacion cumple con funciones de un chat simple.


Para agregar un nuevo contacto debe hacerlo ingresando el userId del contacto a agregar, ejemplo: si tu usuario es leon, tu userId es leonuid  (se debe agregar "*uid*" seguido del usuario, sin espacios)
